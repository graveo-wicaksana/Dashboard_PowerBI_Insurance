Portfolio Project in May 2026: Medical Insurance Cost Dashboard<br>

Built an interactive Power BI dashboard to analyze insurance charges based on:<br>
✔ Age<br>
✔ BMI<br>
✔ Smoking Status<br>
✔ Number of Children<br>
✔ Region<br>

Main findings:<br>
📌 Smokers have substantially higher insurance charges<br>
📌 Obesity contributes heavily to medical expenses<br>
📌 Older age groups tend to produce higher total charges<br>

Source of dataset: https://www.kaggle.com/datasets/mosapabdelghany/medical-insurance-cost-dataset<br>
Accessed at: 10-May-2026<br>
