Portfolio Project in May 2026: Medical Insurance Cost Dashboard

Built an interactive Power BI dashboard to analyze insurance charges based on:
✔ Age
✔ BMI
✔ Smoking Status
✔ Number of Children
✔ Region

Main findings:
📌 Smokers have substantially higher insurance charges
📌 Obesity contributes heavily to medical expenses
📌 Older age groups tend to produce higher total charges

Source of dataset: https://www.kaggle.com/datasets/mosapabdelghany/medical-insurance-cost-dataset
Accessed at: 10-May-2026

Feedback and suggestions are welcome.
